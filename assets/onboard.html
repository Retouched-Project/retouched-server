Secure Setup

Browsers require a trusted connection to enable sensitive features like Haptic Vibration and Motion Sensors.

Initial Step

Download the certificate below. Choose Allow or Save if prompted.

Download Certificate
iOS Instructions
1. Open Settings and tap Profile Downloaded (or go to General > VPN & Device Management).

2. Tap Install and confirm.

3. Go to Settings > General > About > Certificate Trust Settings.

4. Find Retouched Local Root CA and toggle it ON.
Android Instructions
1. Open Settings and search for CA Certificate.

2. Select Install from storage or Encryption & credentials.

3. Choose CA Certificate and tap Install Anyway if warned.

4. Select the RetouchedRootCA.pem file you just downloaded.

On other platforms, install the certificate into your browser or OS trusted root store.
